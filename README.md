# uADSR
Small Envelope Generator based on ATMEGA
Intended for eurorack modular synthesizers

## Parts list (TBF)

- ATMEGA328P
- 28 PIN DIP
- 16MHz Crystal
- 10K Resistor x 9
- 1K Resistor x 2
- 220 Resitor x 1
- 1N4001 Diode x 4
- LED 330 x 1
- Cap 22pF x 2
- TL081 U2 x 1 (OpAmp) *
- MCP4811 x 1 (DAC)
- 16 PIN IDC x 1

Note: Opamp possibly use 741, LM358, TL071?
